# Arthus & Hubert website

Boilerplate from : xxhomey19/bootstrap-webpack-jquery-boilerplate
Used under MIT Licence

## Development

Run development page on **localhost:8080**

```
$ npm run dev
```

## Build

Build for production.

```
$ npm run build
```

## Deploy

Deploy to `gh-pages` branch on GitHub.

```
$ npm run deploy
```
